# Atividade: Controle de LED com Botão no Arduino

Este repositório contém a resolução da atividade prática de **Sistemas Embarcados e Automação**. O desafio consistia em desenvolver um sistema utilizando o simulador **Tinkercad**, onde o acionamento de um botão físico controla o estado de um LED, aplicando conceitos de leitura de entradas digitais e acionamento de componentes.

## 🎯 Objetivos do Projeto

* Praticar o desenvolvimento de sistemas utilizando a plataforma Arduino.
* Aplicar conceitos de programação embarcada em linguagem C++.
* Compreender a leitura de entradas digitais e o controle de saídas para automação.

## 💻 Conteúdo da Entrega

Para a conclusão desta atividade, foram desenvolvidos:
* **Design do Circuito:** Montagem visual com Arduino, LED, Botão e Resistores.
* **Código C++:** Lógica estruturada para o controle do componente de saída.
* **Simulação:** Validação prática do funcionamento no ambiente Tinkercad.
